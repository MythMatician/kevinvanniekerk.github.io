TV Show App

App is built with HTML, CSS some Bootstrap and Javascript with Axios library. Here I used another free API to target an endpoint and return tv shows based on what the user searches for. The event listener is on the button which listens for a submit and then fetches data from the endpoint. I then had to loop through the data and populate the divs on the page with the data that came back from the endpoint. It was a very fun and challenging app for me to build.

Calculator App

App is built with HTML, CSS also some Bootstrap and Javascript. The numbers and operators are represented by buttons and the 'screen' on the top is just a div which acts as our placeholder for the user operations. I looped through the buttons and put a click event listener on each button which grabs the value and appends it into the div on top. When you press the equals button it grabs the value out of the div block and evaluates the string and returns the value to the user in the same block. Very cool app to build.
